# Output Template

Use this structure for the final answer after the intake is complete and all year-specific rules and line numbers have been verified from official sources.

## 1. Tax estimate

Include:
- tax year
- filing type and main forms likely required
- summary of main income sources
- summary of main deductions
- taxes already withheld or prepaid
- estimated refund or estimated amount due
- short note identifying any items that could still change the estimate

## 2. Calculation breakdown

Use clear subsections:
- employment income
- other taxable income
- progression-relevant but tax-free benefits
- capital income treatment
- deductions and allowances
- taxes already paid

State assumptions explicitly. If the calculation still depends on one unresolved value, stop and ask for that value instead of guessing.

## 3. Elster field mapping

Present the mapping in a consistent table or bullet list with these columns or labels:
- form
- section in elster
- exact line number or field label for the selected year
- value to enter
- source or justification
- notes

Examples of forms that may appear:
- est 1 a / hauptvordruck
- anlage n
- anlage vorsorgeaufwand
- anlage kap
- anlage so
- anlage s
- anlage g
- anlage eür
- anlage v
- anlage aus
- other year-relevant forms only if needed

## 4. Auto-import and manual-entry notes

For each certificate-based value that is usually transferred electronically, say:
- whether it is likely to be prefilled
- that the user should verify rather than duplicate it

## 5. Items not to enter

List relevant items that should not be entered and explain why. Examples:
- already taxed domestic capital gains that do not need anlage kap unless a correction is requested
- non-deductible private costs
- values already claimed in another year
- expenses reimbursed by the employer

## 6. Sources

End with the official sources used for:
- legal rule or threshold
- elster field mapping and line numbers
- treatment of special cases

Cite the official source next to each year-sensitive statement.
