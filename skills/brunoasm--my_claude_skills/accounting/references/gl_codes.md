# GL Code Reference

GL (General Ledger) codes categorize expenses in the Field Museum procurement card system. This list includes ALL codes from the budget tab, even those with $0 budget — any code may be used for an expense.

## Entertainment Codes (require supplement form)

| GL Code | Category |
|---------|----------|
| 6455 | Entertainment - Client/Vendor/Partner |
| 6460 | Entertainment - Donor |
| 6470 | Entertainment - Internal Staff Meeting |
| 6475 | Entertainment - Employee Gift |

Any expense using one of these codes requires a **Procurement Card Reconciliation Supplement** to be filed for that month. If alcohol was purchased, Area VP approval is also required.
