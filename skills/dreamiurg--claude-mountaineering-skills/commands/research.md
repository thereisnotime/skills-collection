Research the specified mountain peak and generate a comprehensive route beta report.

If the user provided a peak name as an argument (e.g., `/mountaineering:research Mt Baker`), use that as the target peak. Otherwise, ask which peak to research.

Execute the full route-researcher workflow by invoking the `mountaineering:route-researcher` skill via the Skill tool.
