How it works

ln-100 orchestrates the full pipeline → ln-110 auto-detects project type (backend, frontend, devops) → ln-111–ln-115 generate type-specific docs → ln-120–ln-140 add reference docs, task management, and test strategy → ln-160 extracts procedural content into standalone skills.
