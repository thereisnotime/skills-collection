Prose

Running text sits in a column no wider than layout.measure, so a line never outruns the eye. Headings step the type scale by scale.ratio; links carry the accent; the reading face comes from font.body.

Why size is a token but placement is not

How large a heading runs is a value, so it lives in type.h1 through type.caption. Where an illustration sits relative to its text is a pattern, so it lives in prose, not the token file. The line between the two is the whole discipline.

Guidance

Note

A theme override is a delta, not a copy. The ink theme restates four groups (colours, the three font roles, texture, glow) and inherits the rest of the base unchanged.

  1. Define the neutral base

    Roles + structure, no brand.

  2. Write a theme override

    Only what differs.

  3. Merge and export

    One CSS file per theme.

Colophon

Three files make a theme.

A neutral base.tokens.json, one override that states only the difference, and one merge that joins them. The page you are reading is two such builds, paper.css and ink.css, behind a single switch.
