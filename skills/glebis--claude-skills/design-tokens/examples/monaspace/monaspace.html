§02 — the ideaOne family, five voices

SUPERFAMILY

Monaspace is a monospaced superfamily: five faces that share one metric grid yet read in five registers — from neo-grotesque to handwriting.

Because the cell width is constant, you can mix faces inline without the layout shifting a pixel — a comment in Radon beside code in Neon.

The token set captures this as five fontFamily tokens plus a type group that assigns Xenon to display, Neon to body, Argon to code.

Texture healing nudges narrow glyph pairs apart so the grey value of a block of text evens out — the calm you feel reading it.

§04 — in useThe reason it exists: code

Monaspace Neon with ligatures + texture healing — the working face for the type.code role.

# resolve a brand, then carry it into generation
from dtokens import resolve, export_prompt

resolved = resolve.resolve(model.load("monaspace.tokens.json"))
theme    = export_prompt.to_tufte_theme(resolved, "Monaspace")

if contrast(ink, canvas) >= 7.0 && roles != 0:
    ship(theme)   => "AAA, by construction"

Note the =>, >=, !=, && ligatures and the even rhythm — that's texture healing.
