# 新浪财经实时行情 API 速查

## 端点

```
http://hq.sinajs.cn/list=<codes>
```

- **协议**：HTTP（非 HTTPS）
- **编码**：GBK
- **认证**：无
- **Referer**：需要 `https://finance.sina.com.cn`（反爬基础检查）

## 股票代码格式

| 代码首字符 | 前缀 | 示例 |
|-----------|------|------|
| 6 | sh | 600276 → sh600276 |
| 5 | sh | 512100 → sh512100 |
| 0, 2, 3 | sz | 300122 → sz300122 |
| 4, 8, 9 | sz | 400001 → sz400001 |

## 返回格式

每条一行：
```
var hq_str_<prefix><code>="<32 个逗号分隔字段>"
```

## 一次请求多只股票

逗号分隔，最多约 50 只：
```
http://hq.sinajs.cn/list=sh600276,sz300122,sh600436
```

## 涨跌幅计算

```
涨跌幅% = (现价 - 昨收) / 昨收 × 100
```

注意：停牌/未上市的股票现价可能为 0 或空，需要判断 `prev_close > 0`。

## 非交易时段行为

A 股交易时间 9:30-15:00。非交易时段返回的是上一个交易日的收盘数据，涨跌幅接近 0。

## 已知坑

1. **名称空格**：部分股票名称（如新和成）返回"新 和 成"，需要归一化
2. **编码必须 GBK**：UTF-8 解码会乱码
3. **不支持 HTTPS**：用 HTTP，否则某些网络环境不通
4. **无 CORS**：脚本端不受影响，浏览器端用不了
