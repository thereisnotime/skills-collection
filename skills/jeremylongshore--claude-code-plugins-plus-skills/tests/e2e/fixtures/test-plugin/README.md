# Test Plugin

Minimal test plugin for E2E validation of Claude Code plugin system.

## Overview

This plugin is used exclusively for end-to-end testing of:
- Plugin installation and uninstallation
- Skill loading and activation
- Command parsing
- Metadata validation

## Components

### Skills

- **test-skill** - Basic skill with trigger phrases for activation testing

### Commands

- **/test-command** - Simple command for command parsing tests

## Usage

This plugin is not intended for production use. It exists solely to support the E2E test suite.
