## From the Build Log

We've been building in the open since day one — 414 plugins, 2,788 skills, and a marketplace that ships daily. Now we're bringing the technical writing home.

This blog is where we'll publish deep-dives on AI agent patterns, Claude Code plugin development, and the engineering decisions behind Tons of Skills. Every post also lives on [startaitools.com](https://startaitools.com) and cross-posts to Dev.to and Hashnode with canonical links back to the source.

### What to expect

- **Technical deep-dives** — architecture decisions, debugging war stories, performance wins
- **Plugin development guides** — patterns that work at scale across 400+ plugins
- **AI agent patterns** — what we've learned building with Claude Code daily
- **Build logs** — real-time documentation of features as we ship them

Welcome aboard.
