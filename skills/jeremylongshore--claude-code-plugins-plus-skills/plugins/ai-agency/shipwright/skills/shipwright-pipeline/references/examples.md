# Shipwright Examples

## Build a SaaS Dashboard

```
User: Build me a project management dashboard with team collaboration,
      task boards, and real-time updates.
Stack: Next.js + Supabase
```

Shipwright will generate a complete Next.js application with Supabase auth, a Kanban board UI, real-time subscriptions via Supabase Realtime, and a full test suite.

## Build a Blog

```
User: Create a developer blog with MDX support, syntax highlighting,
      RSS feed, and dark mode.
Stack: Astro
```

Shipwright will scaffold an Astro site with MDX integration, Shiki syntax highlighting, an RSS endpoint, and a theme toggle.

## Enhance an Existing Project

```
User: Add a user settings page with profile editing and notification preferences.
Command: /shipwright:enhance
```

Shipwright will analyze the existing project, add the settings route, update the navigation, create the forms, and add tests for the new feature.
