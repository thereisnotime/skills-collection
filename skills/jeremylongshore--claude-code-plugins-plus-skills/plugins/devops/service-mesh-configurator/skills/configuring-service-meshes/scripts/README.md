# Scripts

Bundled resources for service-mesh-configurator skill

- [x] validate_config.py: Validates the generated service mesh configuration against best practices and common errors.
- [x] deploy_config.sh: Deploys the generated service mesh configuration to the target environment (Kubernetes).
- [x] rollback_config.sh: Rolls back the service mesh configuration to the previous version in case of issues.
- [x] generate_test_traffic.py: Generates test traffic to verify the service mesh configuration.
