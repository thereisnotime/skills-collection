# Jeremy Genkit Terraform

Terraform infrastructure specialist for deploying Firebase Genkit applications to production on Firebase Functions, Cloud Run, and GKE with AI monitoring and auto-scaling.

## Installation

```bash
/plugin install jeremy-genkit-terraform@claude-code-plugins-plus
```

## Features

✅ Firebase Functions deployment with Genkit 1.0+
✅ Cloud Run containerized deployments
✅ GKE cluster deployments
✅ AI monitoring dashboards
✅ Auto-scaling configuration
✅ CI/CD integration
✅ Secret management

## Quick Start

"Deploy my Genkit application to Cloud Run with Terraform"

The skill auto-activates and provisions:
- Cloud Run service with Genkit runtime
- Secret Manager for API keys
- Cloud Monitoring dashboards
- IAM permissions
- Auto-scaling policies
