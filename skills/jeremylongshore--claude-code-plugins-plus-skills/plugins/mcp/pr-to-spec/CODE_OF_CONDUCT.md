# Code of Conduct

## Our Pledge

We are committed to providing a friendly, safe, and welcoming environment for all contributors.

## Our Standards

- Be respectful and constructive
- Accept feedback gracefully
- Focus on what's best for the project
- Show empathy toward other community members

## Enforcement

Unacceptable behavior may be reported to the project maintainers. All complaints will be reviewed and investigated promptly.

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) v2.1.
