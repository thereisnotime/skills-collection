# Hyperfocus

**ADHD-friendly output formatting for Claude Code.**

Restructures AI responses with evidence-based cognitive accessibility — chunking, visual hierarchy, front-loaded answers, and progressive disclosure.

## Install

Inside Claude Code, run:

```
/plugin marketplace add nextor2k/hyperfocus
/plugin install hyperfocus
```

Or via npx skills:

```bash
npx skills add nextor2k/hyperfocus
```

## Usage

- `/hyperfocus` — activate (default: flow mode)
- `/hyperfocus clean|flow|zen` — switch modes
- `stop hyperfocus` — deactivate

## Modes

- **clean** — light touch, professional tone, shorter paragraphs
- **flow** (default) — full structure, What → Why → How, recap sentences
- **zen** — maximum scaffolding, TL;DR at top, lists over prose
