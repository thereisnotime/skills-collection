# PM AI Partner

Structured thinking partner for Product Managers. 12 agent skills, 6 workflow commands, 3 automation hooks.

## Install

```bash
ccpi install pm-ai-partner
```

## Skills (12)

| Category | Skills |
|----------|--------|
| Agent Modes (6) | Thought Partner, Technical Analyst, Devil's Advocate, Writer, Builder, Data Analyst |
| Scenarios (4) | Product Brief, Meeting Prep, Stakeholder Update, Strategic Clarity |
| Analysis (2) | Hypothesis Tester, Competitor Analyst |

## Commands (6)

- `/pm:plan-week` — Weekly planning with context
- `/pm:write-brief [feature]` — Guided product brief creation
- `/pm:prep-meeting [topic]` — Meeting prep with Q&A
- `/pm:audit-codebase [system]` — PM-friendly codebase exploration
- `/pm:competitor-scan [competitor]` — Competitive landscape analysis
- `/pm:metrics-review` — Structured metrics review

## Hooks (3)

- Quality gate for polished documents
- Draft status tracking
- Commit reminders
