# Finta Cost Tuning

## When to Use Free vs Pro

**Stay on Free if:**
- Early-stage exploration, not actively raising
- Small investor list (<20 investors)
- Single founder managing the process

**Upgrade to Pro when:**
- Actively fundraising with 50+ investors in pipeline
- Need multiple deal rooms for different investors
- Want payment collection via Stripe/ACH
- Multiple team members managing the fundraise
