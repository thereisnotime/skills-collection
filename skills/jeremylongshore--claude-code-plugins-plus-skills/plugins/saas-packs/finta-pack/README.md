# Finta Skill Pack

> 18 production-grade Claude Code skills for startup fundraising with Finta CRM

## What Is Finta?

[Finta](https://www.trustfinta.com) is an AI-powered fundraising CRM built for startup founders and fund managers. The platform provides:

- **Pipeline CRM** with automated stage advancement based on email replies and calendar events
- **Aurora AI** for investor prospecting -- ranks investors by thesis fit and warm intro availability
- **Deal Rooms** with shareable links, view tracking, and version control
- **Data Rooms** for secure due diligence document sharing
- **Payment Collection** via Stripe and ACH integration for collecting committed capital
- **Investor Updates** with auto-populated metrics from Stripe, Mercury, and Brex
- **Warm Introduction Network** identifying 2nd-degree connections to target investors

Finta is a web-based platform at trustfinta.com with email/calendar sync, Zapier integration, and CSV import/export. This skill pack provides fundraising workflow patterns and automation strategies for every stage of the fundraise process.

## Quick Start

### 1. Install the Pack

```bash
/plugin install finta-pack@claude-code-plugins-plus
```

### 2. Create Your Finta Account

Sign up at [trustfinta.com](https://www.trustfinta.com) and complete the onboarding wizard with your company details and fundraise parameters.

### 3. Connect Email and Calendar

Go to **Settings** > **Integrations** and connect Gmail/Outlook for automatic investor communication tracking.

### 4. Build Your Pipeline

Import your investor list via CSV or use Aurora AI to discover relevant investors. Follow `finta-core-workflow-a` for the complete fundraise playbook.
