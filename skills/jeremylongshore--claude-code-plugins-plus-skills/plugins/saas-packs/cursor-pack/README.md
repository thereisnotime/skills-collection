# Cursor Skill Pack

> 30 production-grade skills for Cursor IDE mastery -- real .cursor/rules, real Composer workflows, real @-symbol patterns

Complete Claude Code skill pack for [Cursor](https://cursor.com), the AI-first code editor. Every skill contains real configuration examples, real keyboard shortcuts, real settings paths, and real architectural patterns drawn from Cursor's actual documentation and feature set.

## What's Inside

**30 skills** organized across five tiers, covering everything from first-launch setup to enterprise compliance auditing.
