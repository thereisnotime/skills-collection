## File Descriptions

### 1. SKILL.md
**Purpose:** Install and configure Sentry SDK authentication with DSN setup and initialization for Node.js/Python applications.
**Workflow:** This is the first step when adding Sentry to a project. Run this skill when starting a new Sentry integration to get the SDK installed and properly authenticated.
**Relates to:** Directly precedes `sentry-hello-world` for verification. Provides foundation for all other Sentry skills.

## Summary

This skill handles the initial Sentry SDK installation and authentication configuration. It guides through installing the appropriate SDK package (Node.js or Python), configuring the DSN via environment variables, and initializing Sentry in the application. Use this skill when setting up Sentry for the first time or when onboarding a new service to error tracking.
