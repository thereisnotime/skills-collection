### 1. SKILL.md
**Purpose:** Best-practice Sentry architecture patterns for enterprise integration including project structure, configuration patterns, and distributed tracing.
**Workflow:** Reference when designing Sentry integration architecture. Provides patterns for single applications, microservices, and multi-environment setups.

## Summary

This skill provides best-practice patterns for Sentry integration at scale. It covers project structure for single apps, microservices, and multi-environment setups, centralized configuration modules, global error handling patterns, domain-specific error handlers, distributed tracing architecture, and alerting architecture. Use this skill when designing Sentry integration for enterprise applications.
