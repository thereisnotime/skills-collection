## File Descriptions

### 1. SKILL.md
**Purpose:** Production deployment checklist for Sentry integration covering configuration, source maps, security, performance, alerting, and integrations.
**Workflow:** Run before deploying to production to verify all Sentry configuration is complete and appropriate for production workloads.
**Relates to:** Combines elements from multiple skills. Works with `sentry-ci-integration` for deployment automation.

## Summary

This skill provides a comprehensive production-readiness checklist for Sentry integration. It covers configuration verification, source map upload and verification, security settings, performance monitoring setup, alert rule configuration, integration setup, and rollback procedures. Use this skill before any production deployment to ensure Sentry is properly configured for production use.
