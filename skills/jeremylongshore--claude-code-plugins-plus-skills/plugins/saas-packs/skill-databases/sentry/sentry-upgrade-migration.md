## File Descriptions

### 1. SKILL.md
**Purpose:** Upgrade Sentry SDK and migrate between versions, handling breaking changes between major versions in JavaScript and Python.
**Workflow:** Use when upgrading Sentry SDK versions. Review changelog, handle breaking changes, test functionality, and plan rollback.
**Relates to:** Independent upgrade skill. Works with `sentry-common-errors` for post-upgrade issues.

## Summary

This skill guides through upgrading Sentry SDK versions and handling breaking changes. It covers version checking, upgrade process, major version migrations (v6 to v7, v7 to v8), migration checklists, testing after upgrade, and rollback procedures. Use this skill when upgrading Sentry SDK to ensure a smooth transition with no monitoring gaps.
