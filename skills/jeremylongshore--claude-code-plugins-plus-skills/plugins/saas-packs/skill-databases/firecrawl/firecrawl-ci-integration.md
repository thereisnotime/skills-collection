## File Descriptions

### 1. SKILL.md
**Purpose:** Configure FireCrawl CI/CD integration with GitHub Actions, automated testing, and scheduled crawl jobs.
**Workflow:** DevOps skill - integrates scraping pipelines into CI/CD workflows for automated testing and deployment.
**Relates to:** Enables automation for all skills; foundational for firecrawl-deploy-pipeline

## Summary

This skill sets up CI/CD integration for FireCrawl scraping pipelines. It covers GitHub Actions workflows for testing scrape logic, mock fixtures for CI testing without API calls, integration tests with live API (gated by secrets), scheduled crawl jobs using cron triggers, and deployment automation. The integration enables automated testing of scraping code and scheduled data collection workflows.
