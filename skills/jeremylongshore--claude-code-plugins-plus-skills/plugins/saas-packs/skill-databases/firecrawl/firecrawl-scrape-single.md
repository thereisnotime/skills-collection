## File Descriptions

### 1. SKILL.md
**Purpose:** Scrape a single URL and convert to clean markdown with metadata extraction, content formatting, and proper error handling.
**Workflow:** Core scraping workflow - the primary operation for extracting content from individual web pages.
**Relates to:** Follows firecrawl-sdk-patterns; leads into firecrawl-crawl-site for multi-page extraction

## Summary

This skill demonstrates comprehensive single-URL scraping with FireCrawl. It covers the full scrape options including page options (waitFor, timeout, headers), extract options for structured data, markdown formatting preferences, and metadata extraction. The output is clean, LLM-ready markdown suitable for RAG pipelines, content analysis, or data extraction workflows.
