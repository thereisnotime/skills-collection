### 1. SKILL.md
**Purpose:** Apply production-ready FireCrawl SDK patterns including singleton client, error handling, retry logic, and response validation.
**Workflow:** Advanced onboarding skill - establishes enterprise-grade patterns for all FireCrawl interactions.
**Relates to:** Builds on firecrawl-hello-world; foundational for firecrawl-reliability-patterns

## Summary

This skill implements production-ready patterns for the FireCrawl SDK. It covers creating a singleton client instance, implementing robust error handling for network issues and API errors, adding automatic retry with exponential backoff, and validating API responses. These patterns ensure reliable scraping operations in production environments and form the foundation for all other FireCrawl integrations.
