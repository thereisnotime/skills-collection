## File Descriptions

### 1. SKILL.md
**Purpose:** Configure Replit for Education with classroom management, assignments, and automated grading.
**Workflow:** Education administration skill - sets up learning environments for students.
**Relates to:** Builds on replit-team-management concepts; integrates with replit-multiplayer-collab for live teaching.

## Summary

This skill configures Replit Teams for Education for classroom use. It covers creating classroom teams and inviting students, designing and publishing assignments with starter code, configuring automated test-based grading (input/output tests), providing inline feedback on student submissions, tracking student progress and completion, setting up project templates for curriculum, enabling multiplayer for live coding demonstrations, managing student permissions and privacy settings, and integrating with LMS platforms like Google Classroom. Replit for Education streamlines coding instruction.
