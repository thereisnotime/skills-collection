## File Descriptions

### 1. SKILL.md
**Purpose:** Use Replit Agent to build applications from natural language prompts with AI-powered code generation.
**Workflow:** AI-first development skill - introduces Replit's flagship AI agent for rapid application building.
**Relates to:** Foundation for replit-agent-advanced; integrates with replit-database-setup and replit-deployments.

## Summary

This skill introduces Replit Agent, the platform's AI-powered application builder. It teaches developers how to describe applications in natural language, guide the Agent through iterative refinement, review and understand generated code, make corrections and improvements, and deploy the finished application. Replit Agent represents the future of development where AI handles implementation while humans focus on requirements and design.
