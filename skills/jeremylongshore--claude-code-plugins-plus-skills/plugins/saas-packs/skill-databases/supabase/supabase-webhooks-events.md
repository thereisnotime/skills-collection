This skill provides comprehensive guidance on handling Supabase webhooks securely. It covers Express.js webhook endpoints, signature verification with timing-safe comparison, event handler patterns, idempotency with Redis, replay attack protection, and webhook testing. Use this skill when implementing webhook endpoints for Supabase events or when building event-driven integrations.
