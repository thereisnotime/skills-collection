## File Descriptions

### 1. SKILL.md
**Purpose:** Apply Supabase security best practices for secrets management, API key storage, access control, and audit logging.
**Workflow:** Critical operational skill. Should be implemented early in development and verified before production.
**Relates to:** Prerequisite for supabase-prod-checklist; foundation for supabase-enterprise-rbac.

## Summary

This skill covers essential security practices for Supabase integrations. It includes environment variable configuration, secret rotation procedures, least privilege access patterns, webhook signature verification, and audit logging implementation. Use this skill when setting up a new Supabase project, auditing existing security configuration, or preparing for security review.
