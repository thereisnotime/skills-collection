## File Descriptions

### 1. SKILL.md
**Purpose:** Install and configure Supabase SDK/CLI authentication. Sets up API key configuration, environment variables, and verifies connectivity to Supabase services.
**Workflow:** Entry point skill for any Supabase integration. Should be the first skill executed when setting up a new project with Supabase.
**Relates to:** Prerequisite for supabase-hello-world, supabase-local-dev-loop, and all other Supabase skills.

## Summary

This skill guides developers through the complete Supabase SDK installation and authentication setup. It covers installing the SDK via npm or pip, configuring API keys through environment variables or .env files, and verifying the connection works correctly. Use this skill when starting a new Supabase integration or when onboarding new team members to an existing Supabase project.
