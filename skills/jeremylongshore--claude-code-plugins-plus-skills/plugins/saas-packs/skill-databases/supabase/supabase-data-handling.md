## File Descriptions

### 1. SKILL.md
**Purpose:** Implement Supabase PII handling, data classification, redaction, retention policies, and GDPR/CCPA compliance patterns.
**Workflow:** Compliance skill for data handling. Essential for applications processing personal data.
**Relates to:** Complements supabase-security-basics; prerequisite for supabase-enterprise-rbac compliance requirements.

## Summary

This skill provides comprehensive guidance on handling sensitive data in Supabase integrations. It covers data classification by sensitivity level, PII detection with regex patterns, data redaction for logging, retention policies with automatic cleanup, GDPR/CCPA compliance including DSAR and right to deletion, and data minimization practices. Use this skill when handling PII or when ensuring regulatory compliance.
