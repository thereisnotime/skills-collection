## File Descriptions

### 1. SKILL.md
**Purpose:** Configure Supabase across development, staging, and production environments with separate API keys, configuration files, and production safeguards.
**Workflow:** First enterprise skill. Essential for teams deploying to multiple environments.
**Relates to:** Builds on supabase-reference-architecture; enables supabase-observability and supabase-incident-runbook.

## Summary

This skill provides comprehensive guidance on multi-environment Supabase configuration. It covers environment strategy, configuration structure with base and per-environment files, environment detection logic, secret management across local/CI/production, environment isolation with production guards, and feature flags by environment. Use this skill when setting up staging environments or implementing environment-specific configurations.
