### 1. SKILL.md
**Purpose:** Execute Supabase major re-architecture and migration strategies using strangler fig pattern with adapter layers, data migration, and traffic shifting.

**Relates to:** Builds on all enterprise skills; leads to supabase-advanced-troubleshooting for complex issues during migration.

## Summary

This skill provides comprehensive guidance for major Supabase migrations. It covers pre-migration assessment with current state analysis, strangler fig pattern with gradual traffic shifting, adapter layer implementation, batch data migration, feature flag-controlled traffic splitting, rollback procedures, and post-migration validation. Use this skill when migrating to Supabase from another platform or performing major architectural changes.
