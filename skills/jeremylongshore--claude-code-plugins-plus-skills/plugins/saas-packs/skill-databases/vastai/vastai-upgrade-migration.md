# vastai-upgrade-migration

> Safely upgrade SDK versions and migrate between instance types without disruption to running workloads

## Directory Structure

```
vastai-upgrade-migration/
├── SKILL.md
└── examples/
    └── migration_script.py
```

## File Descriptions

| File | Type | Purpose |
|------|------|---------|
| SKILL.md | Markdown | Upgrade and migration procedures with rollback strategies and validation steps |
| examples/migration_script.py | Python | Automated migration script with dry-run mode and rollback capability |
