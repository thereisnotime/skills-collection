## File Descriptions

### 1. SKILL.md
**Purpose:** Collect Clay debug evidence for support tickets and complex troubleshooting. Creates a comprehensive bundle with API logs, configuration (redacted), and network diagnostics.
**Workflow:** Support escalation skill. Use when issues require Clay support team involvement.
**Relates to:** Follows clay-common-errors when self-service troubleshooting fails; accelerates support resolution.

## Summary

This skill guides users through collecting comprehensive debug information for Clay support escalation. It covers capturing API request/response logs, extracting relevant configuration (with secrets redacted), running network connectivity tests, documenting reproduction steps, and packaging everything into a support-ready bundle. Essential for getting fast resolution from Clay support.
