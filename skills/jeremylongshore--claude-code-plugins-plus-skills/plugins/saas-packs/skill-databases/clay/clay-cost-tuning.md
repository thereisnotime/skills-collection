## File Descriptions

### 1. SKILL.md
**Purpose:** Optimize Clay costs through credit management, enrichment selection, and usage monitoring. Enables predictable spend and maximum value from Clay credits.
**Workflow:** Cost optimization skill. Use during budget planning and ongoing cost management.
**Relates to:** Supports all Clay integrations; integrates with clay-observability for usage tracking.

## Summary

This skill optimizes Clay credit consumption and costs. It covers analyzing enrichment usage patterns, selecting cost-effective enrichment providers, implementing usage monitoring and alerts, configuring budget limits, optimizing waterfall enrichment for cost, and identifying waste from failed or duplicate enrichments. Target: 30% cost reduction through optimization.
