## File Descriptions

### 1. SKILL.md
**Purpose:** Execute Clay major migration strategies including data migration from other tools and workflow re-architecture. Enables smooth transitions to Clay.
**Workflow:** Migration skill. Use when migrating to Clay from other enrichment platforms.
**Relates to:** Standalone migration skill; integrates with clay-table-from-requirements for new workflows.

## Summary

This skill guides major migrations to Clay. It covers assessing current enrichment tools and workflows, planning migration phases with rollback points, migrating data from competing platforms (Clearbit, ZoomInfo, Apollo), recreating workflows in Clay's table-based system, implementing parallel running during transition, validating data integrity, and completing the cutover. Essential for teams switching to Clay.
