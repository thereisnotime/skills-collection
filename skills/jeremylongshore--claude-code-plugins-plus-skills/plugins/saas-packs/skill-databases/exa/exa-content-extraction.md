### 1. SKILL.md
**Purpose:** Configure Exa content extraction with text, highlights, and full page content options for downstream processing.

## Summary

This skill covers Exa's content extraction capabilities. It demonstrates the contents parameter with options for text (clean extracted text), highlights (relevant snippets with query context), and full page content. Includes best practices for content processing, handling extraction failures gracefully, and optimizing extraction options based on downstream use cases like RAG pipelines or content analysis.
