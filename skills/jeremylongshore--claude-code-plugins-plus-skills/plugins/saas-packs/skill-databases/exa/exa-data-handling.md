## File Descriptions

### 1. SKILL.md
**Purpose:** Implement data handling policies for search queries, result caching, and PII considerations in Exa usage.
**Workflow:** Compliance skill - ensures data handling meets regulatory requirements.
**Relates to:** Follows exa-security-basics; informs enterprise-scaling decisions

## Summary

This skill covers data handling for Exa: query logging policies (what to log, retention periods), PII detection in search queries before sending, result cache data retention and expiration, GDPR/CCPA considerations for stored search data, audit logging for compliance, data classification for cached content, secure deletion procedures, and privacy-preserving analytics on search patterns. Ensures 100% compliance with data handling requirements.
