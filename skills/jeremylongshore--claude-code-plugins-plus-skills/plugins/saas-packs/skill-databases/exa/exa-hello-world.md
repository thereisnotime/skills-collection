## File Descriptions

### 1. SKILL.md
**Purpose:** Create a minimal working Exa example demonstrating basic semantic search patterns with the neural search API.

**Relates to:** Follows exa-install-auth; leads to exa-search-modes for deeper understanding

## Summary

This skill creates a minimal working Exa example. It demonstrates initializing the Exa client, performing a basic semantic search query, handling the response, and displaying results. The example showcases Exa's neural search capabilities in the simplest possible form, giving developers a working foundation to build upon.
