## File Descriptions

### 1. SKILL.md
**Purpose:** Implement search quality measurement with relevance scoring, A/B testing, and quality dashboards for continuous improvement.
**Workflow:** Search quality engineering skill - enables data-driven search optimization.
**Relates to:** Follows exa-search-modes; integrates with exa-ci-integration

## Summary

This skill implements search quality measurement: relevance metrics (NDCG, MRR, Precision@K), user feedback collection (clicks, dwell time), A/B testing framework for query variations, quality dashboard with trend analysis, automated quality regression detection, query performance analysis, benchmark dataset creation, and continuous improvement workflows. Target: 10% improvement in NDCG through iterative query optimization and search mode tuning.
