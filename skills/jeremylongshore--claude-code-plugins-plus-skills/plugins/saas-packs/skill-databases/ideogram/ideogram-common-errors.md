| File | Type | Purpose |

## Summary

**Category:** Onboarding
**Target Audience:** Developers, Support engineers, DevOps engineers
**Trigger Phrases:** ideogram not working, ideogram errors, fix ideogram, ideogram troubleshoot, ideogram api error

This skill provides systematic troubleshooting for common Ideogram API integration issues. It covers authentication failures, rate limit errors, invalid prompt rejections, timeout issues, and response parsing problems. Use this skill when encountering API errors or unexpected behavior to quickly identify and resolve issues.
