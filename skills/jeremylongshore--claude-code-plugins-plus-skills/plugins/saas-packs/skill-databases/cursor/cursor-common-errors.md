## File Descriptions

### 1. SKILL.md
**Purpose:** Provides comprehensive troubleshooting for common Cursor IDE errors including authentication, completion, indexing, API key, extension, and performance issues.

**Workflow:** This skill activates when users encounter problems with Cursor. It provides symptom-based troubleshooting checklists, solutions, and recovery procedures for crashes and data loss.

**Relates to:** cursor-debug-bundle (AI-specific debugging), cursor-indexing-issues (indexing focus), cursor-performance-tuning (performance focus), cursor-api-key-management (API key issues)

## Summary

This essential troubleshooting skill covers solutions for the most common Cursor IDE issues. It addresses authentication failures, completion problems (not appearing, slow, wrong suggestions), indexing errors (failed, stuck, outdated), API key issues (invalid, rate limited), extension conflicts, performance problems (high CPU/memory), chat errors, crash recovery, and log locations for reporting issues.
