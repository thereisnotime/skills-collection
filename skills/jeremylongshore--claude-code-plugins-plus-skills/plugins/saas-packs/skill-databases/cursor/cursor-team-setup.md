## File Descriptions

### 1. SKILL.md
**Purpose:** Sets up Cursor for teams and organizations including plan selection, member management, onboarding, shared configuration, and cost management.

**Workflow:** This skill activates when administrators set up Cursor for teams. It covers plan comparison, initial setup, member management, team configuration, onboarding procedures, workflows, analytics, cost management, and support resources.

**Relates to:** cursor-prod-checklist (production readiness), cursor-sso-integration (enterprise auth), cursor-usage-analytics (team analytics), cursor-privacy-settings (team security)

## Summary

This administrative skill covers complete team and organization setup for Cursor. It includes plan comparison (Free, Pro $20, Business $40, Enterprise), creating team accounts, member management (inviting, roles, removing), shared configuration (.cursorrules, settings.json), onboarding checklists and documentation, team workflows (code review, pair programming), usage analytics dashboard, cost management and optimization, support resources by plan tier, and security considerations.
