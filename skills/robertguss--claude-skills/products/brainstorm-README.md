# Brainstorm Skill

A collaborative brainstorming partner for multi-session ideation projects that
span days or weeks.

## What's Included

| Skill          | Description                                                                                                                                                                                                                          |
| -------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **brainstorm** | Collaborative brainstorming partner for multi-session ideation. Handles session continuity via versioned markdown documents. Supports SaaS products, books, newsletters, business strategies, and any creative/analytical challenge. |

## Installation

### Claude Code (CLI)

1. Copy the `brainstorm/` folder into your Claude skills directory.
2. In your project's `CLAUDE.md` or global `~/.claude/CLAUDE.md`, add a
   reference to the skill.

Alternatively, add the skill path directly to your Claude Code settings so it
loads automatically.

### Claude.ai (Web/Mobile/Desktop)

1. Go to **Settings → Skills**.
2. Upload the `brainstorm.zip` file.
3. The skill activates automatically when you ask Claude to brainstorm or
   ideate.

## How It Works

The brainstorm skill is designed for multi-session projects. It:

- Maintains a versioned brainstorming document across sessions
- Supports structured ideation methods (e.g., SCAMPER, reverse brainstorming,
  mind mapping)
- Works in two modes: **connected** (cross-project awareness) and
  **clean-slate** (fresh start)
- Acts as a genuine intellectual partner — pushes back, asks clarifying
  questions, and builds on your ideas

**Trigger phrases:** "Let's brainstorm", "Help me ideate", "I want to explore
ideas for...", "Think through this problem with me"
