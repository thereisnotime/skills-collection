01 / WHY

Why it matters.

Three principles, applied to every script.

A

Token-efficient by default

Default output is 3–5 lines. Drill into errors, warnings, or full logs only when you need them. Conversations stay focused, costs stay low.

B

Semantic UI navigation

Find elements by meaning — labels, types, IDs — not pixels. Built on the iOS accessibility tree, so tests survive UI redesigns.

C

Production-ready

Class-based scripts, type hints, Black + Ruff CI. Auto-UDID detection. Idempotent device lifecycle. Works with real simulators today.
